# HR-Analytics-Dashboard-7500-Employees-Power-BI-Project
A comprehensive HR analytics dashboard built using Power BI, analyzing workforce distribution, salary trends, attrition patterns, hiring growth, and employee demographics across 7,500 records. This project provides actionable insights into organizational structure, compensation, and workforce behavior using interactive visualizations

## 📌 Project Overview
This project is an end-to-end **HR Analytics Dashboard** developed using **Power BI**, based on a dataset of **7,500 employees**. The dashboard provides deep insights into workforce distribution, salary structures, hiring trends, attrition patterns, and employee demographics.

The goal of this project is to simulate a **real-world HR analytics solution** that helps organizations make **data-driven HR decisions**.


## 🎯 Key Objectives
- Analyze workforce distribution across job roles
- Understand salary trends and compensation structure
- Identify attrition patterns by department and performance
- Track hiring growth over time
- Evaluate employee demographics (age, gender, qualification)
- Monitor leave balance trends


## 📂 Dataset Details
- Total Records: **7500 Employees**
- Key Fields:
  - Employee ID
  - Full Name
  - Gender
  - Age
  - Job Role
  - Department
  - Qualification
  - Education Level
  - Annual Salary
  - Date of Joining
  - Leave Balance
  - Performance Band


## 📊 Dashboard Features & Insights

### 👥 Workforce Distribution
- Headcount across multiple job roles
- Major workforce concentration in:
  - Operations
  - Engineering
  - Salary 

### ⚖️ Gender Distribution
- Male: ~57%
- Female: ~43%
➡️ Shows relatively balanced workforce diversity

### 🎂 Age Distribution
- Majority workforce falls between **25–40 years**
➡️ Indicates a young and mid-level workforce


### 💰 Salary & Compensation Analysis
- Average salary across roles: ~₹8.8 LPA
- Highest salaries observed in:
  - Product & Engineering roles
- Includes:
  - Min Salary
  - Max Salary
  - Avg Salary

### 🎓 Qualification Impact
- Salary comparison based on:
  - B.Tech, M.Tech, BSc, MSc, BCA, etc.
➡️ Helps understand education vs earning trends

### 📉 Attrition Analysis
- Attrition Rate by:
  - Department
  - Performance Band (High / Medium / Low)

➡️ Key Insight:
- Higher attrition seen in certain departments with low performance bands


### 📈 Hiring Trend & Growth
- Cumulative headcount over time
- Hiring growth from **2010 to 2025**
➡️ Shows steady organizational expansion

### 🏆 Top Paid Employees
- Displays top 5 highest-paid employees
➡️ Useful for executive compensation analysis

### 🏖️ Leave Balance Analysis
- Average leave balance by job role
- Employees with leave balance > 20 days
➡️ Helps HR track employee utilization and burnout risk

## 🛠️ Tools & Technologies Used
- **Power BI Desktop**
- Data Modeling
- DAX (Measures & Calculations)
- Data Visualization Techniques

## 📌 Key Business Insights
- Workforce is heavily concentrated in operational and technical roles
- Salary distribution is consistent but peaks in product-based roles
- Attrition is influenced by performance and department
- Hiring trend shows continuous company growth
- Leave balance indicates potential workforce utilization gaps
